# Fitness Club Website

Учебный проект сайта фитнес-клуба в стиле Invictus, но с чёрно-оранжевой цветовой темой.  
Реализован на чистом HTML и CSS без фреймворков.

## Цели проекта
- Отработать вёрстку без библиотек  
- Использовать слоистую архитектуру CSS  
- Реализовать адаптивную вёрстку  
- Разделить структуру, логику и оформление

## Стек
- HTML5  
- CSS3 (Grid, Flexbox, Media Queries)  
- Без JS и фреймворков

## Структура проекта

/css
reset.css — сброс стандартных стилей браузера
layout.css — сетка страницы, header/main/footer, grid/flex
style.css — цвета, кнопки, карточки, эффекты, тема

/pages
index.html — главная страница
clubs.html — страница клубов
trainers.html — страница тренеров
discounts.html — страница скидок и акций
feedback.html — форма отправки заявки


## Архитектура CSS

Слоистый подход:

1. reset.css  
   - убирает стандартные отступы  
   - нормализует теги  
   - задаёт box-sizing  

2. layout.css  
   - контейнеры  
   - grid и flex  
   - структура страницы (header, main, footer)  

3. style.css  
   - цветовая схема (чёрно-оранжевая тема)  
   - кнопки, карточки, формы  
   - анимации и ховеры  

## Страницы

### Главная

## Адаптивность

Реализован mobile-first:

- 1 колонка на телефоне  
- 2 колонки на планшете  
- 3 колонки на десктопе  
- меню упрощается на маленьких экранах
